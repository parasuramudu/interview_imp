git cammands:
=============
1 to see the list of all local brances

  git branch

2 to see the list of local and remote branchs

  git branch -a

3 to create a new branch
  
  git branch branch_name

4 to move into branch
  
  git checkout or switch branch_name

5 to create a branch and also move into it

  git checkout -b branch_name

6 to merge a branch

  git merge branch_name

7 to delete a branch that is merged (soft delete)
  
  git branch -d branch_name

8 to delete a branch that is not merged (hard delete)
 
  git branch -D branch_name
